Methodology

  • Both logs were evaluated against the same destination host.
  • Nayatel RTT values were parsed from ICMP ping lines in milliseconds.
  • PTCL RTT values were parsed from timestamped CSV sample rows using the destination RTT column.
  • Percentile latency values (P95/P99) were used to evaluate consistency under worst-case moments.
  • Jitter proxy used: mean absolute difference between consecutive RTT samples.

Support Follow-up Focus
